# Mini Bash Toolkit

##  Description
A simple and fun **terminal-based toolkit** written entirely in **Bash**.  
This toolkit provides a few handy utilities like system info, quotes, dice rolling, and a countdown timer â€” all in one script.

##   Features
- View **System Info** (hostname, disk usage)
- Get a **Random Quote**
- **Roll a Dice**
- Run a **Countdown Timer**
