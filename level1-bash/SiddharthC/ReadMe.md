# Level 1 – Bash Basics

Simple Bash scripts covering the basics of shell scripting.

---

## Scripts

| Script Name       | Description                                                        | Key Concepts                     |
| ----------------- | ------------------------------------------------------------------ | -------------------------------- |
| **HelloWorld.sh** | Prints “Hello, World!” to the terminal.                            | `echo`, basic output             |
| **Variables.sh**  | Demonstrates usage of variables and user input.                    | variables, `read`, input/output  |
| **UserInput.sh**  | Reads and processes user input from the terminal.                  | `read`, string handling          |
| **IfElse.sh**     | Uses conditionals to make decisions based on input.                | `if`, `elif`, `else`             |
| **Loops.sh**      | Demonstrates different loop types in Bash.                         | `for`, `while`, iteration        |
| **Calc.sh**       | Performs simple arithmetic (+ − × ÷) using command-line arguments. | `$1`, `case`, arithmetic         |
| **GuessGame.sh**  | A fun number guessing game using random numbers.                   | `$RANDOM`, `while`, conditionals |

---

## ⚙️ How to Run

Make all scripts executable:

```bash
chmod +x *.sh
```

Run any script using:

```bash
./HelloWorld.sh
```
